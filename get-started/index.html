FAQ

What does the batch pipeline do?

With the Track all events from your website, mobile app, server side systems, third party systems and any type of connected device, so that you have a record of what happened, when, and to whom. The Snowplow pipeline then processes these events, which means validation, enrichment, and modeling. It then loads that data into your own data warehouse to power sophisticated analytics.

What is possible with the real-time pipeline?

The real-time pipeline does everything the batch pipeline does.

In addition to regular loading of your data into Redshift, we:

  • Load your data real-time into Elasticsearch, for real time reporting and dashboards
  • Publish your event data into an Amazon Kinesis stream, for real time processing of the data in real-time applications (e.g. personalization)
