The event data pipeline

There are two versions of the Snowplow event data pipeline:

2 pipelines

Batch pipeline

Load your event-level data into your data warehouse to perform advanced analytics.

Real-time pipeline

Load your event-level data into your data warehouse, and make that same data available to data-driven applications and dashboards in real-time.

Both pipelines have the same high-level architecture:

Flexible enrichment process.

Snowplow enriches data after it has been collected, but before it’s stored in Amazon S3 or Redshift. This is the best place to join in sensitive data, external data sources, or data that varies in time. We support these enrichments:

  • GeoIP lookup
  • Weather lookup
  • SQL and RESTful API lookup
  • Useragent and referer parsing
  • Campaign and marketing data parsing
  • Currency conversions
